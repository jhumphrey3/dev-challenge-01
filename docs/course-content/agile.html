Agile Overview:

So what is Agile? Agile is a philosophy that helps teams work together. Agile is one of those things that can be a bit difficult to define, it can be seen as a way of thinking.It lays out a set of values that empowers self-organised teams.

Pros of Agile

  1. Learn from failure
  2. Don't waste time planning at beginning
  3. Maximizes work time
  4. Sounds cool

Cons of Agile

  1. Roles may not be strictly defined
  2. Can become disorganzied
  3. Probably not good for something like makeing a building

Some keywords to remember:

  • Scrum( which is a subset of Agile)
  • Agile Manifesto
  • Acceptence testing
